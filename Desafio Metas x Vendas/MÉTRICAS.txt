(x) Qual foi o faturamento de todo o período?


FATURAMENTO TOTAL = SUMX(fVendas,
fVendas[Quantidade] * fVendas[Preço Unitário])

(x) Qual era a meta de faturamento deste período?
META TOTAL = SUMX(dMetas, dMetas[Asia] + dMetas[Europe] + dMetas[North America])



(x) Qual foi o percentual de atendimento da meta?

% ATENDIMENTO META = [FATURAMENTO TOTAL] / [META TOTAL]



(x) Qual foi a categoria com maior faturamento?

CATEGORIA COM > FATURAMENTO = CALCULATE(
    MAX(dSubcategoria[Categoria]),
    TOPN(1, ALL(dSubcategoria[Categoria]), [QUANTIDADE VENDIDOS])
)



(x) O quanto esta categoria representa no total do faturamento, em percentual?

% CATEGORIA TOP NO TOTAL FATURAMENTO = 
DIVIDE (
    CALCULATE (
        SUMX (
            FILTER (
                dMetas,
                dMetas[Categoria] = [CATEGORIA COM > FATURAMENTO]
            ),
            dMetas[Asia] + dMetas[Europe] + dMetas[North America]
        )
    ),
    [META TOTAL],
    0
) * 100


(x) Quais foram as 3 subcategorias com maior faturamento?

3 SUB CATEGORIA COM > FATURAMENTO = CALCULATE(
    MAX(dSubcategoria[Subcategoria]),
    TOPN(3, ALL(dSubcategoria[Subcategoria]), [FATURAMENTO TOTAL])
)
